# Digital Certificate Generator

A modern web application to generate digital certificates for events. 
Admins can sign up, log in, add events with certificate templates, and view participants.
Users can register for events and download personalized certificates.  

---

## **Features**

### Admin Module
- Secure login/signup with password hashing
- Add new events with certificate templates
- Remove existing events
- View past events and participants
- Responsive modern dashboard with cards and forms

### User Module
- Register for available events
- Download personalized certificates
- Certificates are generated on canvas dynamically

### UI/UX
- Modern and responsive design using CSS gradients, styled cards, and forms
- Works on both desktop and mobile screens
- Interactive and intuitive interface

---

## **Technologies Used**
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP (local development)
- **Fonts:** Google Fonts (Poppins)
